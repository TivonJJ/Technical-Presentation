# Slides Pages

这个目录包含了演讲稿的所有页面，每个文件对应一页幻灯片。

## 📂 文件结构

```
pages/
├── 01-今天我们要讨论的核心内容.md
├── 02-LLM-基础概念.md
├── 03-什么是上下文Context.md (已优化高度)
├── ...
└── 29-谢谢.md
```

共 29 个页面文件。

## ✏️ 如何编辑

### 编辑现有页面

1. 直接打开对应的 `.md` 文件
2. 修改内容
3. 保存后运行 `npm run dev` 查看效果

### 添加新页面

1. 在 `pages/` 目录创建新文件，如 `30-新页面.md`
2. 在主文件 `../slides.md` 中添加引用：
   ```markdown
   ---
   src: ./pages/30-新页面.md
   ---
   ```

### 调整页面顺序

在 `../slides.md` 中调整 `src` 引用的顺序即可。

## 📋 页面格式

每个页面文件包含：

```markdown
---
layout: default  # 或 center, end 等
---

# 页面标题

页面内容...
```

## ✅ 优势

- **易于维护**：每个页面独立，修改不影响其他页面
- **性能更好**：编辑器打开小文件更快
- **便于协作**：多人可以同时编辑不同页面，减少冲突
- **快速定位**：通过文件名快速找到目标页面
- **版本控制友好**：Git diff 更清晰
